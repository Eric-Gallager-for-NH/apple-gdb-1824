The Release Manager's Role

  • The GDB Maintainers appoints a Release Manager.
  • The Release Manager is responsible for organizing, scheduling, and managing releases of GDB.
  • The Release Manager decides the approval and commit policies for release branches, and can change them as needed.
  • The Release Manager can resign at any time. The GDB Maintainers will appoint a replacement in a reasonable amount of time.
  • The GDB Maintainers retain the authority to overrule or replace the Release Manager, but will not ordinarily do so.
